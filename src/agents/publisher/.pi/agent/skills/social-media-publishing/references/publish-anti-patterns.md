# Publish-Time Anti-Patterns

## AP1: Publishing without spoken CTA
Type A (resource listicle) and Type C (tool demo) content must include a
spoken save CTA in the video itself. Caption-only CTAs are not sufficient.

Evidence: spoken CTA produces 4.53% save rate vs 3.29% without (p=0.0075).
If the content creator omitted the spoken CTA, return for re-recording rather
than publishing without it.

## AP2: Publishing sub-30s resource content
Resource lists and tool demos need time to deliver value. Sub-30s content
averages 1.4% save rate. 60s+ averages 4.9%. Publishing a resource list
in 15 seconds wastes the format.

Exception: opinion/hot take content can be 20-30s. Build journals can be
30-45s. Only resource-dense formats need 50-90s.

## AP3: Cross-posting with watermark
Instagram detects and deprioritizes TikTok watermarks (CapCut watermarks
too). Always re-export clean for Instagram. This is a mechanical check —
no judgment needed, just verify absence of watermark before Instagram publish.

## AP4: Same hashtags every post
Instagram flags repeated identical hashtag sets as spam behavior. Rotate
at least 2-3 tags per post. Keep 1-2 consistent niche tags, rotate the rest.

## AP5: Publishing off-niche content
After a viral post, there is pressure to "try something different." This
causes immediate reach collapse. @christieangelica: 972K views on AI tools,
then 371 views on an off-niche quiz. The algorithm doesn't give a grace
period — the drop is instant.

If content is off-niche, escalate to the orchestrating agent before publishing.
The strategic decision to expand niche is not the publisher's to make.

## AP6: Simultaneous cross-posting
Posting to TikTok and Instagram at the exact same time signals automation.
Space cross-posts by 2-4 hours. Post to primary platform first (TikTok),
then Instagram.

## AP7: Ignoring the engagement window
The first 60 minutes after posting are critical for engagement velocity.
If no one is available to respond to comments in that window, delay the
publish to a time when someone will be. A post at 2 AM with no comment
responses performs worse than the same post at 10 AM with active engagement.

## AP8: Publishing AI content without disclosure
Instagram requires AI-generated content to be labeled with "AI info"
disclosure (May 2026). Unlabeled AI content faces reach suppression and
potential removal. If any part of the content was AI-generated (voiceover,
images, scripts), apply the label. Not doing so is a platform risk.
