# Writing Anti-Patterns — Evidence from 611 Classified Posts

## AP1: Generic AI Hype
"AI is revolutionizing everything" — no specific tool, no actionable step.
Measured: generic posts produce 1.62% save rate. Posts naming specific tools
produce 2.40%. 49% lift from specificity.

Fix: name the tool, show the workflow, give the resource link.

## AP2: News Without Analysis
"[Model] just dropped and it's INSANE" — stale in 24 hours, 1.07% save rate
(news_break hook type). No reason to save because the news is everywhere.

Fix: add analysis layer. "What [model] means for [audience] + 3 things to
try today." Transforms news into reference content.

## AP3: Versus Without Verdict
"Claude vs ChatGPT — let me know in comments which you prefer!" — 0.62%
save rate, worst hook type in the dataset.

Fix: take a position. "Claude wins for coding. ChatGPT wins for writing.
Here's why with examples." Opinion + evidence drives saves and comments.

## AP4: Engagement Bait Without Value
"Like if you agree! Follow for more!" — no save reason. Algorithm
deprioritizes content that generates likes without saves/shares.

Fix: give a reason to save. "Save this list for your next project."
The CTA must reference specific value in the post.

## AP5: Overlong Intros
"Hey everyone, welcome back to my channel, today we're going to talk about..."
Wastes first 3 seconds. 50% of viewers drop before second 4.

Fix: hook in first word. "Five books that changed how I build with AI."
Zero preamble. The hook IS the intro.

## AP6: Off-Niche Content
@christieangelica: 972K views on AI tools video, then 371 views on quiz.
Effect is immediate — not gradual decline but cliff-edge drop. Followers
came for AI tools; quiz content gets zero engagement from that audience;
algorithm reads that as poor content quality.

Fix: every post must be recognizably about the same topic. If you want
to test a new niche, use a separate account.
