# Social Media Content Writing — AI/Tech Niche

Based on 858 analyzed posts, 611 classified, 57 video-analyzed. June 2026.

## Video Script Structure

```
HOOK (0-3s): [hook formula from rankings below]
PROMISE (3-8s): brief statement of what viewer gets
BODY (8-50s): deliver items/steps/insights
  - Item 1 (3-10s): [specific name] + [one sentence why]
  - Item 2 (3-10s): ...
  - Item 3 (3-10s): ...
CTA (last 5-10s): spoken save/DM CTA
```

## Hook Formulas (ranked by save rate)

1. "Here are the [N] [things] that will [outcome]" — 4.54%
2. "The [N] best [category] for [audience]" — 2.73%
3. "How to [achieve X] in [timeframe]" — 1.97%
4. "Stop [doing X]. Do this instead." — 1.71%
5. "I [built/launched/hit] X" — 1.52%

NEVER use: "X vs Y which is better?" — 0.62%, worst performer.
AVOID: "[Model] just dropped" — 1.07%, stale fast.

## Writing Rules

### Specificity required
BAD: "Use AI tools to boost your productivity"
GOOD: "Use Claude Code to scaffold a full Next.js app in 10 minutes"
Every script must name at least one specific tool, book, or resource.

### Spoken CTA required
For Type A/C content, include one of:
- "Save this so you don't lose it"
- "Bookmark this for later"
- "Comment [keyword] and I'll DM you the resources"
Spoken CTAs produce 38% higher save rates (p=0.0075).

### Duration targets
- Resource lists: 60-90s
- Tool demos: 30-60s
- Build journals: 30-45s
- Opinions: 20-30s

## Anti-Patterns — Never Write These

For detailed examples, read `{baseDir}/references/writing-anti-patterns.md`.

1. **Generic AI hype** — "AI is changing everything" with no specific tool
2. **News without analysis** — "[Model] just dropped and it's INSANE"
3. **Versus without verdict** — fence-sitting kills engagement
4. **Engagement bait without value** — "Like if you agree!"
5. **Overlong intros** — "Hey everyone, welcome back..." Start with value
6. **Off-niche content** — causes immediate reach collapse after viral posts

## Platform Notes

### TikTok
- On-screen text indexed for search — say AND show keywords
- First 3s decisive, 50% drop-off before second 4
- DM CTA format: "comment [keyword] and I'll DM you"

### Instagram Reels
- Sends (DM shares) weigh more than TikTok
- 5 hashtag limit — keywords in caption matter more
- Carousel format for listicles (high saves, not available on TikTok)

### Cross-posting
- Record for TikTok, re-export without watermark for Instagram
- Add 1-2 carousels/week on Instagram only
