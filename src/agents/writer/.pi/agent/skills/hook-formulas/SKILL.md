# Hook Formulas — Dev Niche Performance Rankings

Hook types ranked by measured save rate from analysis of 532 posts in the AI/tech/developer niche.

## Rankings

| Rank | Hook Type | Save Rate | Pattern | Best For |
|------|-----------|-----------|---------|----------|
| 1 | numbered_promise | 4.54% | "Here are the N things that will..." | Resource lists, tool compilations |
| 2 | listicle | 2.73% | "The N best [category] for [audience]" | Curated recommendations |
| 3 | how_to | 1.97% | "How to [achieve X] in [timeframe]" | Tutorials, workflows, demos |
| 4 | contrarian | 1.71% | "Stop doing X. Do this instead." | Tool comparisons, workflow improvements |
| 5 | personal_story | 1.52% | "I built/launched/hit X" | Build-in-public, milestone posts |
| 6 | news_break | 1.07% | "[Thing] just dropped" | Time-sensitive, stales fast |
| 7 | versus | 0.62% | "X vs Y — which is better?" | Worst performer, avoid |

## When to Use Each

**numbered_promise** — default choice for resource-dense content. Works across all platforms. Use when you have 3+ specific tools/resources to share.

**how_to** — best for tutorials and workflow content. Implies actionable value. Include a timeframe ("in 10 minutes") to increase specificity.

**contrarian** — high engagement driver but requires genuine substance. The "instead" must be specific and defensible. Never use for clickbait — the contrarian claim must be supported by the content.

**personal_story** — builds connection and trust. Use for build-in-public content. Lead with the specific outcome or failure, not "I want to share..."

**news_break** — use sparingly. Stales within 24-48 hours. Only for genuinely significant releases or shifts. Never: "[Model] just dropped and it's INSANE."

**versus** — avoid. Lowest performer. If comparison is needed, frame as contrarian ("Stop using X. Here's why Y is better.") for 2.7x the save rate.

## Platform-Specific Adjustments

### TikTok
- First 3 seconds are decisive — 50% of viewers drop before second 4
- Hook must be spoken AND shown as on-screen text
- Numbered_promise hooks outperform all others on TikTok specifically
- Caption hook (first 80 chars) should reinforce but not duplicate video hook

### X (Twitter)
- First tweet IS the hook — must work standalone in a feed of competing content
- numbered_promise and contrarian work best in tweet format
- Character limit forces compression — every word must earn its place
- Thread hook must promise enough value to justify 5-7 tweet commitment

### LinkedIn
- Hook-first 2-liner format — first 2 lines visible before "see more"
- Professional framing required — contrarian hooks need diplomatic phrasing
- personal_story hooks resonate strongly on LinkedIn (builder narrative)
- Data-backed hooks ("I analyzed 847 posts...") establish credibility fast

### Instagram
- Caption hook (first 125 chars) must work for both Reels and carousels
- Save-optimized hooks (numbered_promise, how_to) outperform on Instagram
- Carousel first slide IS the hook — text must be readable at thumbnail size
- Keywords in first line outperform hashtags for discoverability
